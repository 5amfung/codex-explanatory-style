# Code of Conduct

This project is committed to providing a respectful, harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but prioritize clarity and accountability.
- Give feedback about code and ideas, not about people.
- Help keep the project welcoming to new contributors.

## Unacceptable behavior

- Harassment, threats, or personal attacks
- Discriminatory or demeaning language
- Trolling, intimidation, or sustained disruption
- Publishing private information without permission

## Scope

This code of conduct applies in this repository's issues, pull requests, reviews, and other project spaces.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct. Repeated or severe violations may result in temporary or permanent exclusion from the project space.

## Reporting

If you see behavior that makes the project less safe or less welcoming, open a maintainer-facing report through the repository security reporting flow when appropriate, or contact the maintainer through GitHub.
