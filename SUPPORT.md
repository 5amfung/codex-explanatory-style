# Support

## Getting help

- For bugs, open a GitHub issue with reproduction steps.
- For feature ideas, open a GitHub issue and describe the use case.
- For installation or usage questions, check the README first and then open an issue if the docs are unclear.

## Security issues

For security-sensitive reports, follow [SECURITY.md](SECURITY.md) instead of filing a public issue with full details.

## Scope

This repository is a Codex plugin, so support is focused on:

- plugin installation and marketplace registration
- hook behavior
- explanatory output guidance in this repo

General Codex platform questions may be better answered by the upstream Codex documentation.
